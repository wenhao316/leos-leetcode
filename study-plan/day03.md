# Day 03

## BestTimeToBuySellStock

- Prompt: Find the maximum profit from one buy and one sell of a stock given daily prices.
- Suggested signature: `int maxProfit(int[] prices)`
- Test cases:
  - prices=[7,1,5,3,6,4] -> 5
  - prices=[7,6,4,3,1] -> 0
  - prices=[2,4,1] -> 2

## ProductOfArrayExceptSelf

- Prompt: Return an array where each index has product of all other numbers without using division.
- Suggested signature: `int[] productExceptSelf(int[] nums)`
- Test cases:
  - nums=[1,2,3,4] -> [24,12,8,6]
  - nums=[-1,1,0,-3,3] -> [0,0,9,0,0]
  - nums=[2,3] -> [3,2]

## ContainerWithMostWater

- Prompt: Given heights of vertical lines, return max water container area.
- Suggested signature: `int maxArea(int[] height)`
- Test cases:
  - height=[1,8,6,2,5,4,8,3,7] -> 49
  - height=[1,1] -> 1
  - height=[4,3,2,1,4] -> 16
