# Day 08

## InvertBinaryTree

- Prompt: Invert a binary tree and return root.
- Suggested signature: `TreeNode invertTree(TreeNode root)`
- Test cases:
  - root=[4,2,7,1,3,6,9] -> [4,7,2,9,6,3,1]
  - root=[] -> []
  - root=[2,1,3] -> [2,3,1]

## MaximumDepthOfBinaryTree

- Prompt: Return maximum depth of a binary tree.
- Suggested signature: `int maxDepth(TreeNode root)`
- Test cases:
  - root=[3,9,20,null,null,15,7] -> 3
  - root=[1,null,2] -> 2
  - root=[] -> 0

## DiameterOfBinaryTree

- Prompt: Return diameter (longest path between any two nodes) of a binary tree.
- Suggested signature: `int diameterOfBinaryTree(TreeNode root)`
- Test cases:
  - root=[1,2,3,4,5] -> 3
  - root=[1,2] -> 1
  - root=[] -> 0
